Gitee + Picgo 搭建图床
发表于2020-08-20|更新于2020-08-20|教程
|字数总计:175|阅读时长:1分钟|阅读量:|评论数:

安装Picgo

  • 从官网下载安装 Picgo
  • 需要在插件设置中搜索 gitee 安装 gitee-uploader 插件，因为 giteee 不是默认加载的图床。（ps:安装插件前需要提前安装 node.js)

注册码云（Gitee）创建仓库

在码云网站点击右上角 + 新建仓库

配置 Picgo

点击左侧图床设置，进入 gitee 设置

  • repo : 用户名/创建的仓库名称

  • token : 码云的私人令牌

    1.点击右上角头像，进入设置界面

    2.找到左侧安全设置中私人令牌，点击生成新令牌，把 projects 这一项勾上，其他的不用勾选，然后点击提交。

上传图片，进行验证
